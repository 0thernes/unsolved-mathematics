# The 1/3–2/3 Conjecture

> **Atlas Rank #80** · CSS **49.8** · Field: *Order Theory* · Posed: **1968** · Status: `active-progress`

## Plain statement

Every finite partially ordered set that is not a chain has a pair of elements x, y such that x precedes y in between 1/3 and 2/3 of all linear extensions.
