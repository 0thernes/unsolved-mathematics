# Grothendieck's Section Conjecture

> **Atlas Rank #39** · CSS **64.1** · Field: *Arithmetic Geometry* · Posed: **1983** · Status: `open`

## Plain statement

For a smooth projective curve of genus ≥ 2 over a number field, rational points correspond bijectively to sections of the fundamental exact sequence of étale fundamental groups.
