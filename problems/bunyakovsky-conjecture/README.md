# Bunyakovsky's Conjecture

> **Atlas Rank #19** · CSS **67.3** · Field: *Number Theory* · Posed: **1857** · Status: `open`

## Plain statement

An irreducible integer polynomial of degree >= 2 satisfying simple necessary conditions produces infinitely many prime values.
