# The Eilenberg–Ganea Conjecture

> **Atlas Rank #66** · CSS **56.133** · Field: *Algebraic Topology* · Posed: **1957** · Status: `open`

## Plain statement

Every group of cohomological dimension 2 has geometric dimension 2 (admits a 2-dimensional aspherical CW model).
