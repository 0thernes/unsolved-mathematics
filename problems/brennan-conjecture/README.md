# Brennan's Conjecture

> **Atlas Rank #95** · CSS **53.333** · Field: *Complex Analysis* · Posed: **1978** · Status: `open`

## Plain statement

For a conformal map φ of a simply connected domain, the integral of |φ'|^p is finite for all p in (4/3, 4); the endpoints encode the conjecture.
