# The Jacobian Conjecture

> **Atlas Rank #17** · CSS **68.333** · Field: *Algebraic Geometry* · Posed: **1939** · Status: `open`

## Plain statement

A polynomial map from C^n to C^n with constant nonzero Jacobian determinant has a polynomial inverse.
