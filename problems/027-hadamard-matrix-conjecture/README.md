# The Hadamard Matrix Conjecture

> **Atlas Rank #27** · CSS **57.1** · Field: *Combinatorics* · Posed: **1893** · Status: `active-progress`

## Plain statement

A Hadamard matrix of order 4k exists for every positive integer k.
