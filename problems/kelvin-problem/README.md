# The Kelvin Problem (Optimal Space Partition)

> **Atlas Rank #84** · CSS **55.9** · Field: *Geometry* · Posed: **1887** · Status: `open`

## Plain statement

What partition of space into cells of equal volume minimizes total surface area? Kelvin's truncated octahedron was bettered by the Weaire–Phelan structure (1993), but optimality of neither is proved.
