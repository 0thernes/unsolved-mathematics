# The KPZ Universality Conjecture

> **Atlas Rank #55** · CSS **60.9** · Field: *Probability Theory* · Posed: **1986** · Status: `active-progress`

## Plain statement

A broad class of (1+1)-dimensional random growth models share universal fluctuation exponents (1/3, 2/3) and Tracy–Widom-type limit laws — the KPZ universality class.
