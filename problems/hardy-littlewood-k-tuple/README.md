# The Hardy–Littlewood k-tuple Conjecture

> **Atlas Rank #15** · CSS **69.5** · Field: *Analytic Number Theory* · Posed: **1923** · Status: `active-progress`

## Plain statement

Admissible prime constellations occur infinitely often, with an explicit asymptotic density (the first Hardy–Littlewood conjecture).
