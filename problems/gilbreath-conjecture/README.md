# Gilbreath's Conjecture

> **Atlas Rank #38** · CSS **47.867** · Field: *Number Theory* · Posed: **1958** · Status: `open`

## Plain statement

Iterating absolute differences of the sequence of primes always yields a leading 1 in every row.
