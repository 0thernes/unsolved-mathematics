# The Restriction Conjecture

> **Atlas Rank #24** · CSS **67.667** · Field: *Harmonic Analysis* · Posed: **1967** · Status: `active-progress`

## Plain statement

The Fourier transform of an L^p function can be meaningfully restricted to the sphere (or paraboloid) for an optimal range of p; intimately tied to the Kakeya and Bochner–Riesz conjectures.
