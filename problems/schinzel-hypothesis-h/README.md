# Schinzel's Hypothesis H

> **Atlas Rank #26** · CSS **66.667** · Field: *Number Theory* · Posed: **1958** · Status: `open`

## Plain statement

A finite set of irreducible integer polynomials with no fixed prime divisor takes simultaneous prime values infinitely often; a broad generalization of the twin-prime and Bunyakovsky conjectures.
