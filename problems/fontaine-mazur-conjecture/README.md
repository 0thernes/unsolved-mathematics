# The Fontaine–Mazur Conjecture

> **Atlas Rank #35** · CSS **65.4** · Field: *Number Theory* · Posed: **1995** · Status: `active-progress`

## Plain statement

A geometric p-adic Galois representation (unramified almost everywhere and de Rham at p) arises from algebraic geometry — conjecturally from automorphic forms.
