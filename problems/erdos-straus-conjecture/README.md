# The Erdős–Straus Conjecture

> **Atlas Rank #35** · CSS **52.933** · Field: *Number Theory* · Posed: **1948** · Status: `active-progress`

## Plain statement

For every integer n >= 2, the fraction 4/n is the sum of three unit fractions 1/a + 1/b + 1/c.
