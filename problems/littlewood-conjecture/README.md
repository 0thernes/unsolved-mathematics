# The Littlewood Conjecture

> **Atlas Rank #21** · CSS **66.933** · Field: *Diophantine Approximation* · Posed: **1930** · Status: `active-progress`

## Plain statement

For all real numbers α and β, liminf over n of n·||nα||·||nβ|| = 0, where ||x|| is the distance from x to the nearest integer.
