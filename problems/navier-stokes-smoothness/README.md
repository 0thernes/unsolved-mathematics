# Navier–Stokes Existence and Smoothness

> **Atlas Rank #11** · CSS **74.667** · Field: *Partial Differential Equations* · Posed: **2000** · Status: `open`  · 🏛️ **Millennium Prize Problem**

## Plain statement

In three dimensions, do smooth, globally defined solutions to the incompressible Navier–Stokes equations always exist (and stay smooth)?
