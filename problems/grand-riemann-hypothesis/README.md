# The Grand Riemann Hypothesis

> **Atlas Rank #3** · CSS **84.9** · Field: *Analytic Number Theory* · Posed: **1884** · Status: `open`

## Plain statement

All non-trivial zeros of every automorphic L-function (or every member of the Selberg class) lie on the critical line Re(s) = 1/2; simultaneously generalizes the Riemann and generalized Riemann hypotheses.
