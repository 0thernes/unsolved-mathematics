# The Collatz Conjecture

> **Atlas Rank #16** · CSS **69.267** · Field: *Number Theory / Dynamical Systems* · Posed: **1937** · Status: `open`

## Plain statement

Iterating n -> n/2 (even) or 3n+1 (odd) eventually reaches 1 for every positive integer.
