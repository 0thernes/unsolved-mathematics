# The Telescope Conjecture (Disproved 2023)

> **Atlas Rank #89** · CSS **55.233** · Field: *Algebraic Topology* · Posed: **1984** · Status: `recently-resolved`

## Plain statement

Ravenel's conjecture that the telescopic and K(n)-local localizations of the stable homotopy category agree for n ≥ 2; DISPROVED in 2023, included as a landmark recently-settled problem with rich successor questions.
