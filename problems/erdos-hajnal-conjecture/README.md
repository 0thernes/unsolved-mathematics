# The Erdős–Hajnal Conjecture

> **Atlas Rank #62** · CSS **56.8** · Field: *Graph Theory* · Posed: **1989** · Status: `active-progress`

## Plain statement

Graphs forbidding a fixed induced subgraph contain a clique or independent set of polynomial size n^c, versus the logarithmic size forced in general.
