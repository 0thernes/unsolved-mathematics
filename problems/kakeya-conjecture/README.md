# The Kakeya Conjecture

> **Atlas Rank #17** · CSS **69.1** · Field: *Harmonic Analysis* · Posed: **1971** · Status: `active-progress`

## Plain statement

Every Kakeya set (containing a unit segment in every direction) in R^n has Hausdorff and Minkowski dimension n.
