# The Palis Conjecture (Finitude / Density of Attractors)

> **Atlas Rank #58** · CSS **60.4** · Field: *Dynamical Systems* · Posed: **1995** · Status: `open`

## Plain statement

A typical (positive-probability) dynamical system has finitely many attractors whose basins cover almost all of phase space and which vary statistically stably under perturbation.
