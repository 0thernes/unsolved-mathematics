# Montgomery's Pair Correlation Conjecture

> **Atlas Rank #24** · CSS **61.967** · Field: *Analytic Number Theory* · Posed: **1973** · Status: `open`

## Plain statement

The pair correlation of the non-trivial zeros of the zeta function matches that of eigenvalues of large random Hermitian (GUE) matrices.
