# The Lonely Runner Conjecture

> **Atlas Rank #31** · CSS **55.267** · Field: *Number Theory / Combinatorics* · Posed: **1967** · Status: `active-progress`

## Plain statement

For k runners on a unit circular track with distinct constant speeds, each runner is at some moment at distance >= 1/(k) from all others.
