# Arnold Diffusion (Genericity)

> **Atlas Rank #44** · CSS **62.867** · Field: *Dynamical Systems* · Posed: **1964** · Status: `active-progress`

## Plain statement

In nearly integrable Hamiltonian systems of three or more degrees of freedom, slow drift of the action variables (Arnold diffusion) occurs for a generic, topologically large set of perturbations.
