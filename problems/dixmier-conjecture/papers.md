# Key Papers — The Dixmier Conjecture

_The 25 most important papers. Each carries a verification flag._

The original source is Dixmier's 1968 Weyl-algebra paper (Problème 1). The intellectual center of gravity later shifted to the Jacobian conjecture via the 2005–2007 equivalence; that literature is included because it is, in a precise sense, the same problem. Identifiers are given only where confidently known; "—" marks a real work whose exact DOI/arXiv number is not asserted here. No identifier has been invented.
