# The Dixmier Conjecture

> **Atlas Rank #45** · CSS **62.0** · Field: *Algebra* · Posed: **1968** · Status: `open`

## Plain statement

Every algebra endomorphism of the first Weyl algebra A_1 (over a field of characteristic zero) is an automorphism; stably equivalent to the Jacobian conjecture.
