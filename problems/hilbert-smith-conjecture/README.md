# The Hilbert–Smith Conjecture

> **Atlas Rank #36** · CSS **60.733** · Field: *Geometric Topology* · Posed: **1933** · Status: `active-progress`

## Plain statement

A locally compact group acting faithfully and effectively on a connected topological manifold must be a Lie group; equivalently, no p-adic integer group acts effectively on a manifold.
