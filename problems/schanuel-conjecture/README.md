# Schanuel's Conjecture

> **Atlas Rank #23** · CSS **65.833** · Field: *Transcendental Number Theory* · Posed: **1960** · Status: `open`

## Plain statement

Given n complex numbers linearly independent over Q, the transcendence degree of the field they and their exponentials generate is at least n.
