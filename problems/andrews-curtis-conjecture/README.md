# The Andrews–Curtis Conjecture

> **Atlas Rank #29** · CSS **55.6** · Field: *Geometric Group Theory* · Posed: **1965** · Status: `active-progress`

## Plain statement

Every balanced presentation of the trivial group can be reduced to the trivial presentation by elementary (Andrews–Curtis) moves.
