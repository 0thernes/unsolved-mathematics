# The Zilber–Pink Conjecture

> **Atlas Rank #37** · CSS **64.333** · Field: *Diophantine Geometry* · Posed: **2005** · Status: `open`

## Plain statement

Unlikely intersections of a subvariety with special subvarieties are controlled; a sweeping generalization of the Mordell–Lang and André–Oort conjectures.
