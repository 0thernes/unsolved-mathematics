# The Novikov Conjecture

> **Atlas Rank #22** · CSS **67.5** · Field: *Geometric Topology* · Posed: **1965** · Status: `active-progress`

## Plain statement

The higher signatures of a closed oriented manifold are homotopy invariants, for every discrete group appearing as its fundamental group.
