# The Elliott–Halberstam Conjecture

> **Atlas Rank #36** · CSS **64.0** · Field: *Analytic Number Theory* · Posed: **1968** · Status: `active-progress`

## Plain statement

Primes are equidistributed in arithmetic progressions up to modulus x^(1−ε); a strengthening of the Bombieri–Vinogradov theorem that would sharpen bounded-gaps-between-primes results.
