# The Sunflower Conjecture

> **Atlas Rank #39** · CSS **58.233** · Field: *Extremal Combinatorics* · Posed: **1960** · Status: `active-progress`

## Plain statement

For each k there is C(k) such that any family of more than C(k)^w sets of size w contains k sets forming a sunflower (pairwise intersections all equal).
