# The Zariski Cancellation Problem

> **Atlas Rank #47** · CSS **60.667** · Field: *Algebraic Geometry* · Posed: **1949** · Status: `active-progress`

## Plain statement

If X × A^1 is isomorphic to affine (n+1)-space, must X be isomorphic to affine n-space? Open in characteristic 0; false in positive characteristic.
