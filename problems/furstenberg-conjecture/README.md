# Furstenberg's ×2 ×3 Conjecture

> **Atlas Rank #48** · CSS **61.867** · Field: *Ergodic Theory* · Posed: **1967** · Status: `active-progress`

## Plain statement

The only non-atomic probability measure on the circle simultaneously invariant and ergodic under multiplication by 2 and by 3 is Lebesgue measure.
