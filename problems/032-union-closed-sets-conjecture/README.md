# The Union-Closed Sets Conjecture (Frankl)

> **Atlas Rank #32** · CSS **52.967** · Field: *Combinatorics* · Posed: **1979** · Status: `active-progress`

## Plain statement

In any finite union-closed family of sets (not just the empty set), some element belongs to at least half of the sets.
