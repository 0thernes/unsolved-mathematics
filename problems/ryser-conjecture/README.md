# Ryser's Conjecture (Hypergraph Covering)

> **Atlas Rank #82** · CSS **56.5** · Field: *Combinatorics* · Posed: **1971** · Status: `active-progress`

## Plain statement

In an r-partite r-uniform hypergraph, the covering number is at most (r−1) times the matching number; the r=2 case is König's theorem.
