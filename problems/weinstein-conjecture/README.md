# The Weinstein Conjecture

> **Atlas Rank #45** · CSS **60.967** · Field: *Symplectic Geometry* · Posed: **1979** · Status: `active-progress`

## Plain statement

Every Reeb vector field on a closed contact manifold has at least one periodic orbit; proved in dimension three, open in higher dimensions.
