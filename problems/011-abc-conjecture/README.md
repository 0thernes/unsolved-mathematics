# The abc Conjecture

> **Atlas Rank #11** · CSS **73.767** · Field: *Number Theory* · Posed: **1985** · Status: `disputed-claim`

## Plain statement

For coprime a + b = c, c is bounded in terms of the radical (product of distinct primes) of abc; rad(abc)^(1+ε) dominates c for all but finitely many triples.
