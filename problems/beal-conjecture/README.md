# The Beal Conjecture

> **Atlas Rank #32** · CSS **54.933** · Field: *Number Theory* · Posed: **1993** · Status: `open`

## Plain statement

If A^x + B^y = C^z with x,y,z > 2 and A,B,C positive integers, then A, B, and C share a common prime factor.
