# The Hodge Conjecture

> **Atlas Rank #4** · CSS **79.9** · Field: *Algebraic Geometry* · Posed: **1950** · Status: `open`  · 🏛️ **Millennium Prize Problem**

## Plain statement

On a non-singular complex projective variety, every Hodge class is a rational linear combination of cohomology classes of algebraic cycles.
