# The Caccetta–Häggkvist Conjecture

> **Atlas Rank #45** · CSS **56.733** · Field: *Graph Theory* · Posed: **1978** · Status: `open`

## Plain statement

Every digraph on n vertices with minimum out-degree at least n/k contains a directed cycle of length at most k.
