# The Slice–Ribbon Conjecture

> **Atlas Rank #59** · CSS **57.0** · Field: *Knot Theory* · Posed: **1962** · Status: `open`

## Plain statement

Every smoothly slice knot (bounding a smooth disk in the 4-ball) is a ribbon knot (bounds an immersed disk in the 3-sphere with only ribbon singularities).
