# Köthe's Conjecture

> **Atlas Rank #51** · CSS **59.033** · Field: *Ring Theory* · Posed: **1930** · Status: `open`

## Plain statement

If a ring has no nonzero nil two-sided ideals, it has no nonzero nil one-sided ideals; equivalently, the sum of two nil left ideals is nil.
