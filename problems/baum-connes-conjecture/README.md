# The Baum–Connes Conjecture

> **Atlas Rank #29** · CSS **64.667** · Field: *Operator Algebras* · Posed: **1982** · Status: `active-progress`

## Plain statement

For a locally compact group, the assembly map from the equivariant K-homology of the classifying space to the K-theory of the reduced group C*-algebra is an isomorphism.
