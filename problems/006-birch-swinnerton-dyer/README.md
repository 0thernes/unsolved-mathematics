# The Birch and Swinnerton-Dyer Conjecture

> **Atlas Rank #6** · CSS **78.8** · Field: *Number Theory* · Posed: **1965** · Status: `open`  · 🏛️ **Millennium Prize Problem**

## Plain statement

The rank of the group of rational points of an elliptic curve equals the order of vanishing of its L-function at s = 1.
