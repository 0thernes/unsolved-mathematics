# The Continuum Hypothesis

> **Atlas Rank #9** · CSS **76.2** · Field: *Set Theory & Mathematical Logic* · Posed: **1878** · Status: `independent`

## Plain statement

There is no set whose cardinality is strictly between that of the integers and that of the real numbers. Proven independent of ZFC (Gödel 1940, Cohen 1963); the live question is whether new axioms should decide it.
