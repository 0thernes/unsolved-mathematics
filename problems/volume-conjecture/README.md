# The Volume Conjecture

> **Atlas Rank #56** · CSS **57.467** · Field: *Knot Theory* · Posed: **1997** · Status: `active-progress`

## Plain statement

The asymptotic growth of the colored Jones polynomials of a knot determines the hyperbolic volume of its complement.
