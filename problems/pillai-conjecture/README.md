# Pillai's Conjecture

> **Atlas Rank #61** · CSS **59.767** · Field: *Number Theory* · Posed: **1931** · Status: `open`

## Plain statement

For every fixed k ≥ 1, the equation x^p − y^q = k has only finitely many solutions in integers (x,y,p,q) with p,q ≥ 2; gaps between perfect powers tend to infinity.
