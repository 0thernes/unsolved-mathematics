# The Reconstruction Conjecture (Ulam)

> **Atlas Rank #46** · CSS **56.5** · Field: *Graph Theory* · Posed: **1941** · Status: `open`

## Plain statement

Every finite simple graph on at least three vertices is determined up to isomorphism by its multiset of vertex-deleted subgraphs.
