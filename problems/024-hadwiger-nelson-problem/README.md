# The Hadwiger–Nelson Problem

> **Atlas Rank #24** · CSS **57.3** · Field: *Combinatorial Geometry* · Posed: **1950** · Status: `active-progress`

## Plain statement

How many colors are needed to color the plane so that no two points at distance 1 share a color? Known to be 5, 6, or 7.
