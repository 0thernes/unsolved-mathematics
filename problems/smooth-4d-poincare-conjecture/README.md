# The Smooth 4-Dimensional Poincaré Conjecture

> **Atlas Rank #32** · CSS **65.467** · Field: *Geometric Topology* · Posed: **1982** · Status: `open`

## Plain statement

Every smooth 4-manifold homeomorphic to the 4-sphere is diffeomorphic to it; the last open case of the generalized Poincaré conjecture.
