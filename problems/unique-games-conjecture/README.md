# The Unique Games Conjecture

> **Atlas Rank #30** · CSS **64.033** · Field: *Computational Complexity Theory* · Posed: **2002** · Status: `active-progress`

## Plain statement

For every ε, distinguishing unique-label-cover instances that are (1−ε)-satisfiable from ε-satisfiable is NP-hard; if true, it pins down the exact approximability threshold of many optimization problems.
