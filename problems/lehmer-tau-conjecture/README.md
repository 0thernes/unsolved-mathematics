# Lehmer's Conjecture on the Ramanujan Tau Function

> **Atlas Rank #61** · CSS **56.9** · Field: *Number Theory* · Posed: **1947** · Status: `open`

## Plain statement

The Ramanujan tau function never vanishes: τ(n) ≠ 0 for all positive integers n.
