# Vaught's Conjecture

> **Atlas Rank #50** · CSS **59.267** · Field: *Mathematical Logic* · Posed: **1961** · Status: `open`

## Plain statement

A countable complete first-order theory has either at most countably many or exactly continuum-many countable models — never strictly in between.
