# The Tate Conjecture

> **Atlas Rank #14** · CSS **72.533** · Field: *Arithmetic Geometry* · Posed: **1963** · Status: `open`

## Plain statement

On a smooth projective variety over a finitely generated field, the cycle-class map surjects onto the Galois-invariant étale cohomology classes; an arithmetic analogue of the Hodge conjecture.
