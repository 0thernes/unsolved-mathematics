# Singmaster's Conjecture

> **Atlas Rank #37** · CSS **48.0** · Field: *Combinatorics* · Posed: **1971** · Status: `active-progress`

## Plain statement

There is a finite upper bound on the number of times any integer > 1 can appear in Pascal's triangle.
