# Carmichael's Totient Conjecture

> **Atlas Rank #77** · CSS **56.867** · Field: *Number Theory* · Posed: **1907** · Status: `open`

## Plain statement

For every n there is some m ≠ n with φ(m) = φ(n); i.e. no value of Euler's totient is taken exactly once.
