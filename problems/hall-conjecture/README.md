# Hall's Conjecture

> **Atlas Rank #74** · CSS **53.8** · Field: *Number Theory* · Posed: **1971** · Status: `open`

## Plain statement

For non-square integers, the gap |x^3 − y^2| grows at least like a power x^(1/2−ε); a special, sharp consequence of the abc/Vojta circle of ideas.
