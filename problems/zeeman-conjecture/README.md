# The Zeeman Conjecture

> **Atlas Rank #68** · CSS **55.467** · Field: *Geometric Topology* · Posed: **1964** · Status: `open`

## Plain statement

For every contractible 2-complex K, the product K × [0,1] is collapsible; it implies both the (now proven) 3-D Poincaré conjecture and the Andrews–Curtis conjecture.
