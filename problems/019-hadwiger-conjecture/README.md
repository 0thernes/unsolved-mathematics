# Hadwiger's Conjecture (Graph Coloring)

> **Atlas Rank #19** · CSS **67.067** · Field: *Graph Theory* · Posed: **1943** · Status: `active-progress`

## Plain statement

Every graph with no K_t minor is (t-1)-colorable; a vast generalization of the Four Color Theorem.
