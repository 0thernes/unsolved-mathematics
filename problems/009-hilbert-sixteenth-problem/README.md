# Hilbert's Sixteenth Problem (Second Part)

> **Atlas Rank #9** · CSS **76.033** · Field: *Dynamical Systems* · Posed: **1900** · Status: `open`

## Plain statement

Bound the number and relative position of limit cycles of a planar polynomial vector field of degree n; even for n = 2 no finite bound is known.
