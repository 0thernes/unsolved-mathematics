# The Erdős Unit Distance Problem

> **Atlas Rank #33** · CSS **61.667** · Field: *Combinatorial Geometry* · Posed: **1946** · Status: `open`

## Plain statement

What is the maximum number of unit distances among n points in the plane? Conjectured n^(1+o(1)); the best bounds remain far apart.
