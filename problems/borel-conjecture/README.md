# The Borel Conjecture

> **Atlas Rank #34** · CSS **64.1** · Field: *Geometric Topology* · Posed: **1953** · Status: `active-progress`

## Plain statement

Closed aspherical manifolds with isomorphic fundamental groups are homeomorphic, and any homotopy equivalence between them is homotopic to a homeomorphism.
