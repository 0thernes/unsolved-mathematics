# Giuga's Conjecture

> **Atlas Rank #91** · CSS **54.6** · Field: *Number Theory* · Posed: **1950** · Status: `open`

## Plain statement

An integer n > 1 is prime if and only if the sum of the (n−1)-th powers of 1..n−1 is congruent to −1 modulo n.
