# The Four Exponentials Conjecture

> **Atlas Rank #40** · CSS **58.1** · Field: *Transcendental Number Theory* · Posed: **1944** · Status: `open`

## Plain statement

Given two Q-linearly independent complex numbers x1,x2 and two more y1,y2, at least one of the four exp(xi*yj) is transcendental. (The six exponentials theorem is proved; four remains open.)
