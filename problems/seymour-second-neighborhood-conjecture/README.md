# Seymour's Second Neighborhood Conjecture

> **Atlas Rank #79** · CSS **49.933** · Field: *Graph Theory* · Posed: **1990** · Status: `open`

## Plain statement

Every oriented graph has a vertex whose second out-neighborhood is at least as large as its first out-neighborhood.
