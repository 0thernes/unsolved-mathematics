# The Graceful Tree Conjecture (Ringel–Kotzig)

> **Atlas Rank #54** · CSS **52.267** · Field: *Combinatorics* · Posed: **1964** · Status: `active-progress`

## Plain statement

Every tree admits a graceful labeling: vertices labeled with distinct integers in [0,n] so edge labels (absolute differences) are all distinct.
