# Crouzeix's Conjecture

> **Atlas Rank #38** · CSS **43.9** · Field: *Matrix Analysis* · Posed: **2004** · Status: `active-progress`

## Plain statement

For any square matrix A and polynomial p, ||p(A)|| <= 2 max over the numerical range W(A) of |p|.
