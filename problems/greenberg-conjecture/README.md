# Greenberg's Conjecture

> **Atlas Rank #67** · CSS **58.467** · Field: *Number Theory* · Posed: **1976** · Status: `open`

## Plain statement

For a totally real number field, the Iwasawa invariants λ and μ of the cyclotomic Z_p-extension both vanish.
