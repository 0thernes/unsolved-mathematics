# Brocard's Problem

> **Atlas Rank #26** · CSS **56.733** · Field: *Number Theory* · Posed: **1876** · Status: `open`

## Plain statement

Are there only finitely many integers n for which n! + 1 is a perfect square (Brown numbers)?
