# The MLC Conjecture (Mandelbrot Locally Connected)

> **Atlas Rank #37** · CSS **63.667** · Field: *Complex Dynamics* · Posed: **1982** · Status: `active-progress`

## Plain statement

The Mandelbrot set is locally connected; this would yield a complete topological/combinatorial model of it and imply density of hyperbolicity for quadratic polynomials.
