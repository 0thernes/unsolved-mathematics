# The Bombieri–Lang Conjecture

> **Atlas Rank #25** · CSS **67.5** · Field: *Diophantine Geometry* · Posed: **1980** · Status: `open`

## Plain statement

On a variety of general type over a number field, the rational points are not Zariski dense; a higher-dimensional analogue of the Mordell conjecture.
