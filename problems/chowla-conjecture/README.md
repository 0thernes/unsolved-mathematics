# The Chowla Conjecture

> **Atlas Rank #38** · CSS **63.6** · Field: *Analytic Number Theory* · Posed: **1965** · Status: `active-progress`

## Plain statement

Correlations of the Liouville function λ at distinct shifts average to zero; a quantitative statement of the pseudorandomness of the Möbius/Liouville function.
