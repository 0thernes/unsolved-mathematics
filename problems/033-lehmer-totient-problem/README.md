# Lehmer's Totient Problem

> **Atlas Rank #33** · CSS **54.3** · Field: *Number Theory* · Posed: **1932** · Status: `open`

## Plain statement

Are there any composite numbers n such that Euler's totient φ(n) divides n - 1?
