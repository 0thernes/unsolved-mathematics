# The Falconer Distance Conjecture

> **Atlas Rank #38** · CSS **59.967** · Field: *Geometric Measure Theory* · Posed: **1985** · Status: `active-progress`

## Plain statement

A compact set in R^d with Hausdorff dimension greater than d/2 has a distance set of positive Lebesgue measure.
