# Lehmer's Mahler Measure Conjecture

> **Atlas Rank #35** · CSS **61.333** · Field: *Number Theory* · Posed: **1933** · Status: `open`

## Plain statement

There is an absolute constant gap above 1 in the Mahler measures of integer polynomials: no Mahler measure lies strictly between 1 and some fixed bound (conjecturally Lehmer's number, ~1.17628).
