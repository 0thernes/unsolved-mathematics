# The Catalan–Dickson Conjecture (Aliquot Sequences)

> **Atlas Rank #55** · CSS **57.933** · Field: *Number Theory* · Posed: **1888** · Status: `open`

## Plain statement

Every aliquot sequence (iterating the sum-of-proper-divisors function) is bounded, hence eventually periodic; Guy and Selfridge conjecture the opposite.
